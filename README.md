## `Install Manually ðŸ‘‡`
## `Requirements`
* [Node.js](https://nodejs.org/en/)
* [Git](https://git-scm.com/downloads)
* Any text editor
## `Clone Repo & Installation dependencies`
```bash
apt update & upgrade -y
apt install git -y
pkg install nodejs -y 
git clone https://github.com/wmm-malinda/free-dat-bot
cd free-dat-bot

npm start
```
